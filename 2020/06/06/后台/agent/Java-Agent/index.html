    Java Agent

    java.lang.instrument.Instrumentation(Java Agent API)

    它允许JVM在加载某个class文件之前对应用程序字节码进行修改，同时也支持对已加载的class(类字节码)进行重新加载(Retransform)。


    它可以做什么？

    加载class文件前拦截、修改字节码，运行期已加载类的字节码变更，监控JVM上运行的应用程序，jar热部署 等….

    JVM Tool Interface

    JVM Tool Interface

    JDK1.5开始，Java新增了 JVMTI(JVM Tool Interface) 接口；

    该接口为运行在JVM虚拟机中的应用程序，提供了 检查、控制 等功能。java.lang.instrument 包的实现，就是基于这种机制的；

    Instrumentation 功能的实现，就是基于 JVMTI。

    在 Instrumentation 的实现当中，存在一个 JVMTI 的代理程序，通过调用 JVMTI 当中 Java 类相关的函数来完成 Java 类的动态操作。

    Java Agent 使用

    Java Agent代理程序 与 应用程序 是分开独立的jar包。


    命令行启动 agent：

    -javaagent:jarpath

jarpath是代理JAR文件的路径

    Java Agent jar的 MANIFEST.MF 文件中，必须包含 Premain-Class 属性。


    Java Agent 的入口，是实现一个 premain 的公共静态方法(类似于main())。

    JVM初始化后，premain()方法会被调用。

    premain()方法有两种：

    ① public static void premain(String agentArgs, Instrumentation inst);

    ② public static void premain(String agentArgs);

    JVM首先尝试调用 premain(String agentArgs, Instrumentation inst)，如果没有实现这个方法，那就尝试调用 premain(String agentArgs)。


    MANIFEST.MF 文件配置

    Premain-Class：指定JVM启动时，需要加载的代理类（包含 premain 方法的 全路径类名）。

    Agent-Class：指定虚拟机启动后，需要加载的代理类（包含 agentmain 方法的 全路径类名）。

    Boot-Class-Path：Bootstrap类加载器要搜索的路径列表；

    public class TheClassFileTransformer implements ClassFileTransformer {

    private static final String CLASS_NAME = "org/server";

    private static final String CLASS_NAME_2 = "org.server";

    @Override
    public byte[] transform(ClassLoader loader, String className, Class<?> classBeingRedefined, ProtectionDomain protectionDomain, byte[] classfileBuffer) throws IllegalClassFormatException {

        // 只处理特定类
        if (className.startsWith(CLASS_NAME)) {

            String newClassName = className.replace("/", ".");
            System.out.println("Transforming: " + newClassName);

            ClassPool pool = ClassPool.getDefault();
            CtClass ctClass;
            try {
                pool.insertClassPath(new LoaderClassPath(loader));
                try {
                    ctClass = pool.get(newClassName);
                } catch (NotFoundException e) {
                    // Spring会能过字节码对我们的类进行扩展，这种情况下找不到对应的class文件，会报NotFoundException
                    // 这时我们可以直接读取classfileBuffer，直接通过类的字节码创建CtClass对象
                    ByteArrayInputStream is = null;
                    try {
                        is = new ByteArrayInputStream(classfileBuffer);
                        ctClass = pool.makeClass(is);
                    } finally {
                        if (null != is) {
                            is.close();
                            is = null;
                        }
                    }
                }
                return process(ctClass);
            } catch (Exception e) {
                System.out.println("className:{" + className + "},errMsg:{" + e.getMessage() + "} ,err:{" + e.toString() + "}");
            }
        }
        return null;
    }

    private byte[] process(CtClass ctClass) throws CannotCompileException, IOException {
        if (ctClass.isInterface()) {
            return null;
        }
        CtMethod[] methods = ctClass.getDeclaredMethods();
        for (CtMethod method : methods) {
            enhance(method);
        }
        return ctClass.toBytecode();
    }

    private void enhance(CtMethod method) throws CannotCompileException {
        method.insertBefore("{ System.out.println(\"" + method.getLongName() + " called ...\"); }");

        method.instrument(new ExprEditor() {
            @Override
            public void edit(MethodCall m) {
                try {
                    if (m.getClassName().startsWith(CLASS_NAME_2)) {

                        StringBuilder sb = new StringBuilder();
                        sb.append("{long startTime = System.currentTimeMillis();");
                        sb.append("$_=$proceed($$) + \" amended...\";");
                        sb.append("System.out.println(\"");
                        sb.append(m.getClassName()).append(".").append(m.getMethodName());
                        sb.append(" cost: \" + (System.currentTimeMillis() - startTime) + \" ms\");}");
                        m.replace(sb.toString());
                    }
                } catch (CannotCompileException e) {
                    System.out.println("className:{" + m.getClassName() + "},methodName:{" + m.getMethodName() + "},errMsg:{" + e.getMessage() + "},err:{" + e.toString() + "}");
                }
            }
        });
    }
}

    public class MethodCostTime {

    @RuntimeType
    public static Object intercept(@Origin Method method, @SuperCall Callable<?> callable) throws Exception {
        long start = System.currentTimeMillis();
        try {
            // 原有函数执行
            return callable.call();
        } finally {
            System.out.println(method + " 方法耗时： " + (System.currentTimeMillis() - start) + "ms");
        }
    }

}


public class MyAgent {

    /**
     * JVM 首先尝试在代理类上调用以下方法
     *
     * @param agentArgs
     * @param inst
     */
    public static void premain(String agentArgs, Instrumentation inst) {
        System.out.println("this is my agent：" + agentArgs);

        AgentBuilder.Transformer transformer = (builder, typeDescription, classLoader, javaModule) -> builder
                // 拦截任意方法
                .method(ElementMatchers.any())
                // 委托
                .intercept(MethodDelegation.to(MethodCostTime.class));

        AgentBuilder.Listener listener = new AgentBuilder.Listener() {
            @Override
            public void onDiscovery(String s, ClassLoader classLoader, JavaModule javaModule, boolean b) {
            }
            @Override
            public void onTransformation(TypeDescription typeDescription, ClassLoader classLoader, JavaModule javaModule, boolean b, DynamicType dynamicType) {
            }
            @Override
            public void onIgnored(TypeDescription typeDescription, ClassLoader classLoader, JavaModule javaModule, boolean b) {
            }
            @Override
            public void onError(String s, ClassLoader classLoader, JavaModule javaModule, boolean b, Throwable throwable) {
            }
            @Override
            public void onComplete(String s, ClassLoader classLoader, JavaModule javaModule, boolean b) {
            }
        };

        new AgentBuilder
                .Default()
                // 指定需要拦截的类
                .type(ElementMatchers.nameStartsWith("org.server"))
                .transform(transformer)
                .with(listener)
                .installOn(inst);
    }

    /**
     * 如果代理类没有实现上面的方法，那么 JVM 将尝试调用该方法
     *
     * @param agentArgs
     */
    public static void premain(String agentArgs) {
    }

}
