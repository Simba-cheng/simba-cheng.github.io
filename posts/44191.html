Vue3 指令速查
创建时间:2025-05-20 14:31:46  更新时间:2025-05-23 09:59:06    3k 字    #前端  #🥦 Vue 
This post is also available in English and alternative languages.

引用 & 扩展
* Vue3: directives — cheat sheet (built-in and custom)

在 Vue3 中，指令是带有 v- 前缀的特殊属性，用于将动态效果应用到网页元素上。比如，你可以用指令控制元素的显示、绑定数据或监听事件。Vue3 内置了许多常用指令，还支持开发者创建自定义指令。

1. v-text

v-text 指令用于将 textContent 插入元素，其工作原理与插值（ {{ }} ）完全相同。它们之间的用法区别在于 v-text 会替换元素的整个 textContent ，而插值只允许替换部分内容。


2. v-html

v-html 指令用于将动态 HTML 注入另一个元素 （相当于在 JavaScript 中使用 innerHTML ）。这是一个需要谨慎使用的指令，因为它可能允许 XSS（跨站脚本）攻击 ，从而将恶意代码注入到应用程序的 DOM 中。


3. v-show

v-show 指令根据接收到的值动态改变元素的 display 属性。 在示例中，当 visible 状态为 true 时，段落会收到 display: block （ <p> 元素的默认显示）。点击按钮会将 visible 状态更改为 false ，并将 display: none 赋给段落。


4. v-if / v-else-if / v-else

v-if 、 v-else-if 和 v-else 被认为是 Vue 中最常用的指令，用于组件和元素的动态渲染并遵循与原生 JavaScript 中的 if 、 else if 和 else 相同的逻辑。在示例中，如果 type 为 'A' ，我们将渲染一个段落；但是，如果类型为 'B' ，我们将渲染一个锚点；在任何其他情况下，我们将渲染一个名为 ErrorComponent 的组件。

注意：由 v-if 、 v-else-if 和 v-else 管理的元素会根据满足的条件在 DOM 中完全销毁或重建，而不像 v-show 那样只会改变元素的 display 。


5. v-for

v-for 指令通过迭代数组或对象来呈现元素或组件的列表， 类似于 forEach 。

关于 v-for 重点是，需要提供一个特殊的 :key 属性，该属性必须接收一个唯一的值。这个值可以直接从 tasks 里获取，也可以使用数组的 index （如果需要操作数组中的元素，则不建议使用数组的index，因为这可能会导致错误）。

key属性是用来帮助 Vue 标识列表中的元素，这样 Vue 就不需要在列表变化时重新创建它们。


6. v-on

v-on 指令（或简称为 @ ） 向 HTML 元素添加“事件监听器”， 类似于使用 JavaScript 的 addEventListener 所做的事情。

任何标准 JavaScript 事件都可以与 v-on 指令一起使用，该指令还接受行为修饰符和或键修饰符 。


7. v-bind

v-bind 指令用于创建/绑定动态 HTML 属性到元素或将 props 传递给子组件。 v-bind 指令可以缩写为 : 作为简写。

  • 完整形式：v-bind:属性名=”表达式”，如 v-bind:style=”cardStyle”。
  • 缩写形式：:属性名=”表达式”，如 :style=”cardStyle”。
  • :style="cardStyle"：绑定卡片的内联样式（如背景色）。
  • :class="{ bold: isBold }"：动态绑定类名，控制文本加粗。

8. v-model

v-model 指令创建双向数据绑定，使得输入、选择和组件之间的状态同步变得容易。

  • <input v-model="username">：文本输入框，双向绑定到 username。
  • <input type="checkbox" v-model="showContent">：复选框，双向绑定到 showContent。
  • <p v-if="showContent">：根据 showContent 控制是否显示输入内容。

核心用法：v-model="变量"，在表单元素（如 <input>、<textarea>、<select>）上创建双向数据绑定。用户在输入框或复选框操作时，v-model 自动更新 data 中的变量。当 data 中的变量变化时，Vue 自动更新表单元素的值。


9. v-slot

v-slot 指令用于在组件中定义并使用插槽。 与通过 props 相比， Slots 是一种更灵活地将内容传递给子组件的方式，并且可以命名或不命名，允许父组件向子组件的插槽插入自定义内容，从而增强组件的灵活性和复用性。

  • 子组件通过定义占位符，父组件使用 v-slot 插入具体内容（如 HTML、组件或文本）。

  • 插槽让子组件的结构可由父组件动态定义，类似“模板填充”。

  • 默认插槽 <slot>：用于标题，默认显示“默认标题”。
  • 具名插槽 <slot name="content">：用于正文，默认显示“默认正文”。
  • 按钮通过 @click 绑定 toggleContent，切换 isFirstContent。
  • ContentCard 组件使用 v-slot：
    • 默认插槽：显示动态标题，颜色通过 :style 绑定（蓝色/绿色）。
    • 具名插槽 content：显示动态正文内容。

10. v-pre

用于跳过 Vue 的模板编译过程，直接渲染原始的 HTML 内容。
