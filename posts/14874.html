2. 数据类型

JavaScript 中有五种基本的数据形式：字符串、整数、布尔值、undefined 和 null，又被称为原始数据类型。

2.4. Array

Array Methods

JavaScript 数组具有以下特点：
  1. 首先，数组可以包含混合类型的值。例如，你可以有一个同时存储数字、字符串、布尔值和 null 类型的数组。
  2. 其次，数组的大小是动态的，会自动扩展。也就是说，你不需要在一开始就指定数组的大小。

  • Methods

    • forEach()：可以遍历数组并对每个元素执行一个操作。

    • map()：可以通过对原数组的每个元素执行一个操作来创建一个新数组。

    • filter()：可以通过筛选掉不满足特定条件的元素，生成一个新的数组。

    • reduce()：对数组中的每个元素按序执行一个提供的 reducer 函数，每一次运行 reducer 会将先前元素的计算结果作为参数传入，最后将其结果汇总为单个返回值。

      第一次执行回调函数时，不存在“上一次的计算结果”。如果需要回调函数从数组索引为 0 的元素开始执行，则需要传递初始值。否则，数组索引为 0 的元素将被用作初始值，迭代器将从第二个元素开始执行（即从索引为 1 而不是 0 的位置开始）。

    • find()：返回数组中满足提供的测试函数的第一个元素的值。否则返回 undefined。

      filter() 和 find() 方法的主要区别是，filter() 会返回所有满足条件的元素。

    • findIndex()：返回数组中满足提供的测试函数的第一个元素的索引。若没有找到对应元素则返回 -1。

    • some()：检查数组中是否至少有一个元素通过了由提供的函数实现的测试。如果在数组中找到一个元素使得提供的函数返回 true，则返回 true；否则返回 false。它不会修改数组。

    • every()：检查一个数组内的所有元素是否都能通过指定函数的测试，返回一个布尔值。

    • flat()：创建一个新的数组，并根据指定深度递归地将所有子数组元素拼接到新的数组中。

    • flatMap()：对数组中的每个元素应用给定的回调函数，然后将结果展开一级，返回一个新数组。它等价于在调用 map() 方法后再调用深度为 1 的 flat() 方法（arr.map(...args).flat()），但比分别调用这两个方法稍微更高效一些。

    • slice()：返回一个新的数组对象，这一对象是一个由 start 和 end 决定的原数组的浅拷贝（包括 start，不包括 end），其中 start 和 end 代表了数组元素的索引，原始数组不会被改变。

    • splice()：就地移除或者替换已存在的元素和/或添加新的元素。

      要创建一个删除和/或替换部分内容而不改变原数组的新数组，请使用 toSpliced()。要访问数组的一部分而不修改它，参见 slice()。

    • unshift()：将指定元素添加到数组的开头，并返回数组的新长度。

    • shift()：从数组中删除第一个元素，并返回该元素的值。此方法更改数组的长度。

    • includes()：用来判断一个数组是否包含一个指定的值，根据情况，如果包含则返回 true，否则返回 false。

2.6. null vs undefined

  • undefined：意味着一个变量已经声明，但还没有被赋予任何具体的值。换句话说，它只是一个存在但尚未被赋值的变量。
  • null：null 类似于 undefined，它需要显式地赋值。它同样表示空或者无，不过是因为开发者有意设置成这样的。

3. 作用域

📝 JavaScript 作用域和变量提升

作用域决定了变量的可见性和可访问性。JavaScript 中有三种作用域类型：「块级作用域」、「函数作用域」、「全局作用域」


3.1. 块级作用域

在 ES6（2015）之前，JavaScript 只有 「全局作用域」 和 「函数作用域」。ES6 引入了两个重要的新关键字：let 和 const。这两个关键字为 JavaScript 提供了「块级作用域」。在 { } 块中声明的变量无法从块外部访问。


3.2. 函数作用域

每个函数都会创建一个新的作用域。在函数内部定义的变量在函数外部无法访问。在函数内部用 var、let 和 const 声明的变量都具有相似的作用域行为。


3.3. 全局作用域

在全局范围内声明的变量（不在任何函数内部）具有全局作用域。全局变量可以在 JavaScript 程序中的任何地方访问。在块外部用 var、let 和 const 声明的变量也具有全局作用域。
