🥦 Vue ref属性
创建时间:2023-04-08 15:28:21  更新时间:2025-04-21 09:14:59    1.8k 字    #前端  #🥦 Vue 
This post is also available in English and alternative languages.

引用 & 扩展
* brokyz's blog - Vue2 ref

* Vue3中操作ref的四种使用方式，建议收藏！


ref 是 Vue 中的一个特殊属性，它用于给元素或组件注册一个引用。在 Vue 实例中，可以使用 ref 来引用DOM元素或子组件，并在需要的时候直接访问它们。

比如：在使用 Vue 时，有时会需要拿到 DOM 元素进行操作。这时 Vue 不推荐进行使用原生 JS 获取 DOM 元素。而是提供了 ref 方便去获取组件中的 DOM 元素。

ref 属性是 Vue2 和 Vue3 中都有的，但是使用方式不大一样。


1. Vue2 中的 ref 属性

1.1. 在HTML标签中使用

给 HTML 标签加上 ref 属性并指定属性名，就可以在 Vue 中通过 vm 实例的 $refs 属性获取到对应的 ref 属性值的标签。

  • person.vue（person组件）

  • App.vue


1.2. 在组件标签中使用

在组件标签中添加 ref，通过 $refs 拿到的是子组件的实例对象 VueComponent。

  • person.vue（person组件）

  • App.vue


2. Vue3 中的 ref 属性

2.1. 在模板中使用 ref 访问 DOM 元素

通过对 div 元素添加 ref 属性，为了获取到这个元素，声明了一个与 ref 属性名称相同的变量 sectionRef，然后通过 sectionRef.value 的形式即可获取该 div 元素。

使用场景：单一 DOM 元素或者个数较少的场景。


2.2. 父容器 ref 遍历子元素操作

  • listRef 绑定在 .list-section 父级 div 上，意味着 listRef.value 指向该父级 DOM 元素。
  • v-for 生成多个 div.list-item，用于模拟列表，每个 div 代表一个 list 数据项。
  • @click="higherAction(index)" 监听点击事件，点击后调用 higherAction(index)，用于增加该项高度。

使用场景：通过 v-for 循环生成的固定数量元素的场景。


2.3. 父组件访问子组件

  • 在 ParentComponent.vue 中，<ChildComponent ref="childComponent"/> 绑定了 ref，从而可以在 script setup 中访问子组件实例。
  • 在 ChildComponent.vue 中，使用 defineExpose({ childData, childMethod }) 使父组件可以访问 childData 和 childMethod。
  • onMounted(() => { console.log(childComponent.value); }) 确保组件挂载后，可以正确访问 ref 绑定的子组件实例。
  • handleClick 方法中，childComponent.value.childMethod() 直接调用了子组件的方法，这在 Vue 3 setup 语法下是标准用法。
