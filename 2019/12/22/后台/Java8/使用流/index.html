    流(Stream)的使用

    流(Stream)的使用一般包括下面三件事：

    1. 一个数据源(如集合)来执行一个查询

    2. 一个中间操作链，形成一条流水线。

    3. 一个终端操作，执行流水线，并生成结果。

    流(Stream)的流水线背后的理念类似于构建器模式。

    在构建器模式中有一个调用链用来设置一套配置（对流来说这就是一个中间操作链），接着是调用built方法（对流来说就是终端操作）。

    PS:这段代码是公用代码，后面都会用到

    private static List<Dish> getDish() {
    List<Dish> result = Arrays.asList(
            new Dish("pork", false, 800, Dish.Type.MEAT),
            new Dish("beef", false, 700, Dish.Type.MEAT),
            new Dish("chicken", false, 400, Dish.Type.MEAT),
            new Dish("french fries", true, 530, Dish.Type.OTHER),
            new Dish("rice", true, 350, Dish.Type.OTHER),
            new Dish("season fruit", true, 120, Dish.Type.OTHER),
            new Dish("pizza", true, 550, Dish.Type.OTHER),
            new Dish("prawns", false, 300, Dish.Type.FISH),
            new Dish("salmon", false, 450, Dish.Type.FISH));
    return result;
}

    filter 筛选

    Streams接口支持filter()方法。

    filter()方法接收Predicate<? super T>作为参数，并返回一个包括所有符合谓词的元素的流

    筛选出所有，卡路里小于500的菜

    List<Dish> dishes = getDish();

List<Dish> collect = dishes.stream()
        .filter(dish -> dish.getCalories() < 500)
        .collect(Collectors.toList());

for (Dish dish : collect) {
    System.out.println(dish.toString());
}

    filter distinct 去重

    Stream流还支持distinct()方法

    根据 hashCode 和 equals 进行去重。

    public static void filterDistinctDemo() {

    List<Integer> numbers = Arrays.asList(1, 2, 1, 3, 3, 2, 4);

    numbers.stream()
            .filter(i -> i % 2 == 0)
            .distinct()
            .forEach(System.out::println);
}

    filter limit 限量(截断)

    Stream流还支持 limit(n)方法。

    该方法会返回一个不超过给定长度的流；

    简单地说，就是 限制返回数量。

    limit()只关心限制数量，不会帮我们排序。

    public static void filterLimit() {

    List<Dish> dishes = getDish();

    List<Dish> collect = dishes.stream()
            .filter(dish -> dish.getCalories() < 500)
            .limit(2)
            .collect(Collectors.toList());

    for (Dish dish : collect) {
        System.out.println(dish.toString());
    }
}

    filter skip 跳过

    skip(n)方法，返回一个扔掉前n个元素的流。

    如果流中元素不足n个，则返回一个空流。

    筛选卡路里 大于 300的菜，并且是头两个。

    public static void filterSkip() {

    List<Dish> dishes = getDish();

    List<Dish> collect = dishes.stream()
            .filter(dish -> dish.getCalories() > 300)
            //.skip(20) //返回空流
            .skip(2)
            .collect(Collectors.toList());

    for (Dish dish : collect) {
        System.out.println(dish.toString());
    }
}

    skip 和 limit 的区别

    这两个方法都是截取流，但是他们有一些区别：

    skip

    skip 操作必须时刻监测流中元素的状态，才能判断是否需要丢弃。

skip 属于状态操作。

    limit

    limit 只关心截取的是不是其参数 maxsize（最大区间值），其它毫不关心。一旦达到就立马中断操作返回流。

limit 属于一个中断操作。

    映射(map)

    流支持map方法，它会接收一个函数作为参数；

    这个函数会被应用到每个元素上，并将其映射成一个新的元素。

    （使用映射一词，是因为它和转换类似，但其中的细微差别在于它是"创建一个新版本"而不是去"修改"）

    从集合中获取所有的菜名，并放在List集合中：

    public static void getNames() {
    List<Dish> dishes = getDish();
    List<String> names = dishes.stream().map(Dish::getName).collect(Collectors.toList());
    for (String name : names) {
        System.out.println(name);
    }
}

    这种写法很爽啊，换做以前，要先循环 dishes ，然后挨个getName()，将其放在一个临时List容器中。

    再看一个例子，获取集合中，每个字符串的长度

    public static void getNameLength() {
    List<Dish> dishes = getDish();
    List<Integer> nameLengths = dishes.stream()
            .map(Dish::getName)
            .map(String::length)
            .collect(Collectors.toList());
    System.out.println(nameLengths);
}

    flatMap

    flatMap流的扁平化

    筛选两个词中，不同的字符

    /**
 * 筛选出两个词中，不同的字符
 */
public static void getDistinct() {
    String[] arrayOfWords = {"Goodbye", "World"};
    List<String> collect = Arrays.stream(arrayOfWords)
            .map(s -> s.split(""))
            .flatMap(Arrays::stream)
            .distinct()
            .collect(Collectors.toList());
    System.out.println(collect);
}

    使用flatMap方法的效果是，让你把一个流中的每个值都换成一个流，然后把所有的流连接起来成为另一个流。

    检查

    anyMatch、allMatch、noneMatch 三个方法都用到了所谓的短路;

    anyMatch

    流(Stream)中只要有一个元素符合传入的 predicate，返回 true

    菜肴中是否包含素菜

    public static void testAnyMatch() {
    List<Dish> dishes = getDish();
    boolean anyMatch = dishes.stream().anyMatch(Dish::isVegetarian);
    if (anyMatch) {
        System.out.println("The menu is (somewhat) vegetarian friendly!!");
    }
}

    allMatch

    流(Stream)中全部元素符合传入的 predicate，返回 true

    菜肴中的菜品是否有利健康(卡路里小于1000)

    public static void testAllMatch() {
    List<Dish> dishes = getDish();
    boolean allMatch = dishes.stream().allMatch(dish -> dish.getCalories() < 1000);
    System.out.println(allMatch);
}

    noneMatch

    和allMatch相对的是noneMatch

    流(Stream)中没有一个元素符合传入的 predicate，返回 true

    菜肴中的菜品没有 卡路里大于1000的

    public static void testNoneMatch() {
    List<Dish> dishes = getDish();
    boolean noneMatch = dishes.stream().noneMatch(dish -> dish.getCalories() >= 1000);
    System.out.println(noneMatch);
}

    查找

    findAny

    查找任何一个就返回 Optional，一个容器对象，该对象可以包含一个null或非null值。

    /**
 * 在菜肴列表中找到一个荤菜就返回
 */
public static void findAnyTestApp() {
    List<Dish> dishes = getDish();

    dishes.stream()
            //筛选条件
            .filter(Dish::isVegetarian)
            //查找到一个就返回（短路）
            .findAny()
            //当值不为null时，执行代码段
            .ifPresent(dish -> System.out.println(dish.toString()));
}

    如果返回值不为null，则调用isPresent()方法将返回true

    public static void findAnyTestApp2() {
    List<Dish> dishes = getDish();

    boolean present = dishes.stream()
            //筛选条件
            .filter(Dish::isVegetarian)
            //查找到一个就返回（短路）
            .findAny()
            //如果值存在，则返回true
            .isPresent();

    System.out.println(present);
}

    或者

    public static void findAnyTestApp3() {
    List<Dish> dishes = getDish();

    Optional<Dish> optionalDish = dishes.stream()
            //筛选条件
            .filter(Dish::isVegetarian)
            //查找到一个就返回（短路）
            .findAny();

    if (optionalDish.isPresent()) {
        System.out.println(optionalDish.get());
    }
}

    findFirst

    有些List或排序好的数据，生成的流；对与这种流，如果想要找到第一个元素，可以使用findFirst();

    举个栗子：

    找出列表中第一个平方能被3整除的数

    /**
 * 找出列表中第一个平方能被3整除的数
 */
public static void findFirstTestApp() {

    List<Integer> someNumbers = Arrays.asList(1, 2, 3, 4, 5);

    Optional<Integer> first = someNumbers.stream()
            //元素进行平方
            .map(num -> num * num)
            //过滤值是否整除
            .filter(num -> num % 3 == 0)
            //返回第一个
            .findFirst();

    System.out.println(first.get());
}

    reduce求和

    public static void reduceTestApp() {
    List<Integer> someNumbers = Arrays.asList(1, 2, 3, 4, 5);

    Integer reduce = someNumbers.stream().reduce((integer, integer2) -> integer * integer2).get();

    max、min(最大值、最小值)

    计算最大值、最小值，只需要将Integer.min、Integer.Max方法传递给Lambda

    public static void maxOrMinTestApp() {
    List<Integer> someNumbers = Arrays.asList(-1, -98, 1, 2, 3, 4, 5, 0, 45, 11, 63, 845, 74);

    Integer minResult = someNumbers.stream().reduce(Integer::min).get();
    System.out.println("min : " + minResult);

    Integer maxResult = someNumbers.stream().reduce(Integer::max).get();
    System.out.println("max : " + maxResult);
}

    实战-Demo

    public class TestApp {

    public static void main(String[] args) {
        List<Transaction> transactions = getValue();

        //2011年所有交易并按照交易额排序(从高到低)
        List<Transaction> transactionResult = transactions.stream()
                //筛选年份
                .filter(transaction -> 2011 == transaction.getYear())
                //根据交易额排序
                .sorted(Comparator.comparing(Transaction::getValue))
                //转换List
                .collect(Collectors.toList());
        System.out.println("transactionResult:" + transactionResult);

        //交易员在哪些城市
        List<String> cityResults = transactions.stream()
                //提取城市信息
                .map(transaction -> transaction.getTrader().getCity())
                //去重
                .distinct()
                .collect(Collectors.toList());
        System.out.println("cityResults:" + cityResults);

        Set<String> setCityResult = transactions.stream()
                //提取城市信息
                .map(transaction -> transaction.getTrader().getCity())
                .collect(Collectors.toSet());
        System.out.println("setCityResult:" + setCityResult);

        //查找所有剑桥的交易员，按照姓名排序（效率不高,所有字符串被反复连接,每次都要创建一个新的String）
        List<Trader> traderResults = transactions.stream()
                //从交易信息中提取交易员
                .map(Transaction::getTrader)
                //筛选剑桥
                .filter(trader -> "Cambridge".equals(trader.getCity()))
                //去重
                .distinct()
                //姓名排序
                .sorted(Comparator.comparing(Trader::getName))
                .collect(Collectors.toList());
        System.out.println("traderResults:" + traderResults);

        //可以使用joining()（内部使用StringBuilder）
        String resultTraderStr = transactions.stream()
                .map(transaction -> transaction.getTrader().getName())
                .distinct()
                .sorted()
                .collect(Collectors.joining(","));
        System.out.println("resultTraderStr:" + resultTraderStr);

        //交易员是否有在米兰工作的
        boolean milan = transactions.stream()
                .anyMatch(transaction -> transaction.getTrader().getCity().equals("Milan"));
        System.out.println("milan:" + milan);

        //打印生活在剑桥交易员的交易额
        List<Integer> transactionValues = transactions.stream()
                .filter(transaction -> "Cambridge".equals(transaction.getTrader().getCity()))
                .map(Transaction::getValue)
                .collect(Collectors.toList());
        System.out.println("transactionValues:" + transactionValues);

        //筛选交易中，最高的交易额
        Integer maxValue = transactions.stream()
                .map(Transaction::getValue)
                .reduce(Integer::max).get();
        System.out.println("maxValue:" + maxValue);

        //筛选交易额最小的交易
        Transaction minValue = transactions.stream()
                .min(Comparator.comparing(Transaction::getValue)).get();
        System.out.println("minValue:" + minValue);
    }

    private static List<Transaction> getValue() {
        Trader raoul = new Trader("Raoul", "Cambridge");
        Trader mario = new Trader("Mario", "Milan");
        Trader alan = new Trader("Alan", "Cambridge");
        Trader brian = new Trader("Brian", "Cambridge");
        List<Transaction> transactions = Arrays.asList(
                new Transaction(brian, 2011, 300),
                new Transaction(raoul, 2012, 1000),
                new Transaction(raoul, 2011, 400),
                new Transaction(mario, 2012, 710),
                new Transaction(mario, 2012, 700),
                new Transaction(alan, 2012, 950)
        );
        return transactions;
    }
}

    数值流

    先看一个demo，使用reduce方法，计算流中元素的总和。

    计算菜单中所有菜品卡路里总量：

    public static void test_1() {
    List<Dish> dishes = getDish();
    Integer reduceSum = dishes.stream()
            .map(Dish::getCalories)
            .reduce(0, Integer::sum);
    System.out.println("reduceSum:" + reduceSum);
}

    上面这段代码，暗含一个 “装箱成本” ; 每个Integer都必须拆箱成原始类型，在进行求和。

    数值流(原始类型特化流)

    Java8中引入了三个’原始类型-特化流’接口来解决这个问题：IntStream，DoubleStream，LongStream；
    以此避免暗含装箱的成本；这三个接口又可以叫做’数值流’

    public static void testMapToInt() {
    List<Dish> dishes = getDish();

    //求卡路里总和
    int calorieSum = dishes
            //获取一个Stream<Dish>
            .stream()
            //获取一个IntStream
            .mapToInt(Dish::getCalories)
            .sum();
    System.out.println("calorieSum:" + calorieSum);

    //求菜品中卡路里最大的
    int max = dishes.stream().mapToInt(Dish::getCalories).max().getAsInt();
    System.out.println("max:" + max);

    //求菜品种卡路里最小的
    int min = dishes.stream().mapToInt(Dish::getCalories).min().getAsInt();
    System.out.println("min:" + min);

    //菜品中卡路里大于600的数量
    long count = dishes.stream()
            .mapToInt(Dish::getCalories)
            .filter(value -> value > 600)
            .count();
    System.out.println("count:" + count);
}

    数值流 -> 普通流

    但，一旦有了数值流，你可能想把它转换回非特化流。

    例如，IntStream上只能产生原始int型整数，但是你可能想生成另一类值，可以使用boxed方法

    public static void testMapToInt2() {
    List<Dish> dishes = getDish();
    //将Stream转换为数值流IntStream
    IntStream intStream = dishes.stream().mapToInt(Dish::getCalories);
    //将数值流转换成Stream一般流
    Stream<Integer> boxed = intStream.boxed();
    Integer maxReduce = boxed.reduce(Integer::max).get();
    System.out.println("maxReduce:" + maxReduce);
}

    OptionalInt设置默认值

    public static void testOrElse() {
    //筛选卡路里最大值
    OptionalInt optionalInt = getDish().stream().mapToInt(Dish::getCalories).max();
    //手动模拟，没有最大值(空)
    optionalInt = OptionalInt.empty();
    //设置默认值
    int max = optionalInt.orElse(53);
    System.out.println("max:" + max);
}

    数值范围

    生成1 ~ 100之间所有数字。

    在 IntStream 和 LongStream 中提供了 range() 和 rangeClosed() 供我们使用。

    它们之间的区别：range()不包含结束值，rangeClosed包含结束值。

    public static void testRangClosed() {
    IntStream intStream = IntStream
            //生成从1-100的偶数
            .rangeClosed(1, 100)
            //链接filter方法，筛选出偶数
            .filter(value -> value % 2 == 0);
    //符合的偶数计数
    long count = intStream.count();
    System.out.println("count:" + count);
}

    构建流

    之前都是使用stream()方法，从集合生成流；下面尝试从值序列、数组、文件来创建流

    值创建流

    使用静态方法Stream.of()显示的创建一个流；
    Stream.of()方法，可以接受任意数量的参数。

    public static void createStreamTest() {
    //通过显示值，创建一个流
    Stream<String> stringStream = Stream.of("Java 8", "Lambdas", "In", "Action");
    stringStream
            //对流中的每个参数，进行toUpperCase()操作
            .map(String::toUpperCase)
            //输出
            .forEach(System.out::println);
}

    数组创建流

    使用Arrays.stream静态方法，从数组创建一个流。

    它接受一个数组作为参数,将原始类型int的数组转换成一个IntStream

    public static void createArrayStreamTest() {
    int[] numberArr = {1, 6, 4, 2, 8, 99, 54, 3};
    int sum = Arrays.stream(numberArr).sum();
    System.out.println("sum:" + sum);
}

    文件生成流

    使用NIO中,Files.lines方法,它会返回指定文件中每行数据，组成的字符串流。

    Files.lines()方法得到的流，其中每个元素都是指定文件中的一行

    public static void createFileStreamTest() throws Exception {
    /*
        data.txt:
            12345678
            123
            234
            456
     */
    //从文件中读取所有行，作为流(流会自动关闭)
    Stream<String> lines = Files.lines(Paths.get("./data/data.txt"), Charset.defaultCharset());

    List<Long> longList = lines
            //每个元素去空格
            .map(String::trim)
            //元素转换成long型
            .map(Long::parseLong)
            //筛选(元素为偶数)
            .filter(aLong -> aLong % 2 == 0)
            //填充到list
            .collect(Collectors.toList());
    System.out.println("longList:" + longList);
}
