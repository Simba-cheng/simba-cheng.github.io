    е…ій”®еӯ—multi_matchпјҢйҖҡеёёз”ЁеңЁжҹҘиҜўеӨҡдёӘfieldsж—¶гҖӮ

    fieldsеұһжҖ§жҢҮе®ҡйңҖиҰҒжҹҘиҜўзҡ„еӯ—ж®өгҖӮ

    еңЁnewsзҙўеј•пјҢnewзұ»еһӢдёӢпјҢжүҖжңүеӯ—ж®өдёӯпјҢжҹҘиҜўеёҰжңүвҖҷжұҹиӢҸвҖҷзҡ„з»“жһң

{
  "query": {
    "multi_match": {
      "query": "жұҹиӢҸ"
    }
  }
}

    еңЁnewsзҙўеј•пјҢnewзұ»еһӢдёӢпјҢжүҖжңүеӯ—ж®өдёӯпјҢжҹҘиҜўеёҰжңүвҖҷ18вҖҷзҡ„з»“жһң
