    行为参数化，它是可以帮助你处理频繁变更需求的一种开发模式，可让代码更好地适应不断变化的要求，减轻未来的工作量

    ‘行为参数化’就是传递代码，一个方法接受多个不同的行为作为参数，并在内部使用它们，完成不同行为的能力。

    Demo

    上手几个demo，切实体会下

    参照《Java8实战》中的例子：

    Demo-1

    筛选绿色苹果

    public static void main(String[] args) {
    List<Apple> apples;
    List<Apple> appleList = getApples();
    apples = filterApples(appleList, Apple::isGreenApple);
    for (Apple apple : apples) {
        System.out.println("greenApple:" + apple.toString());
    }
}

public static List<Apple> filterApples(List<Apple> inventory, Apple.Predicate<Apple> p) {
    List<Apple> result = new ArrayList<>();
    for (Apple apple : inventory) {
        if (p.filter(apple)) {
            result.add(apple);
        }
    }
    return result;
}

private static List<Apple> getApples() {
    List<Apple> appleList = new ArrayList<>();
    for (int i = 0; i < 20; i++) {
        Apple apple = new Apple();
        apple.setAppleName("烟台红富士" + i);
        apple.setWeight(Integer.parseInt("100" + i));
        if (i % 2 == 0) {
            apple.setColor("red");
        } else {
            apple.setColor("green");
        }
        appleList.add(apple);
    }
    return appleList;
}

    public interface ApplePredicate {
    boolean filter(Apple apple);
}

public class AppleRedAndHeavyPredicate implements ApplePredicate {
    @Override
    public boolean filter(Apple apple) {
        return "red".equals(apple.getColor()) && apple.getWeight() > 1010;
    }
}

public static void main(String[] args) {
    List<Apple> appleList = getApples();
    List<Apple> apples = filterApples(appleList, new AppleRedAndHeavyPredicate());
    for (Apple a : apples) {

}

    Demo-5

    行为参数化，Lambda表达式，自定义函数接口，抽象化并引入泛型。

    public interface Analysis<T> {
    boolean analysis(T t);
}

public class FruitAnalysis {
    public static <T> List<T> filter(List<T> list, Analysis<T> p) {
        List<T> result = new ArrayList<>();
        for (T t : list) {
            if (p.analysis(t)) {
                result.add(t);
            }
        }
        return result;
    }
}

public static void main(String[] args) {
    List<Apple> appleList = getApples();
    List<Apple> appleResult = FruitAnalysis.filter(appleList, (Apple a) -> "red".equals(a.getColor()));
    for (Apple a : appleResult) {
        System.out.println(a.toString());
    }
    System.out.println();
    appleResult = FruitAnalysis.filter(appleList, (Apple a) -> a.getWeight() > 10010);
    for (Apple a : appleResult) {
        System.out.println(a.toString());
    }
}

private static List<Apple> getApples() {
    List<Apple> appleList = new ArrayList<>();
    for (int i = 0; i < 20; i++) {
        Apple apple = new Apple();
        apple.setAppleName("烟台红富士" + i);
        apple.setWeight(Integer.parseInt("100" + i));
        if (i % 2 == 0) {
            apple.setColor("red");
        } else {
            apple.setColor("green");
        }
        appleList.add(apple);
    }
    return appleList;
}

    行为参数化，Lambda表达式在其他地方的使用

    public static void main(String[] args) {

    //老的写法
    Thread thread = new Thread(new Runnable() {
        @Override
        public void run() {
            System.out.println("1-2-3-4-5");
        }
    });
    thread.start();

    //使用lambda，简化内部类写法
    Thread thread1 = new Thread(() -> System.out.println("2-4-6-8-10"));
    thread1.start();
}
